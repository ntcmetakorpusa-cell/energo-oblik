# ⚡ ЕнергоОблік

Веб-застосунок для обліку показань лічильників електроенергії та холодної води.

## Можливості

- 📊 17 лічильників електроенергії + 7 лічильників холодної води
- 👥 Ролі: Адміністратор, Електрик, Директор, Бухгалтер
- 📱 Адаптивний дизайн (мобільний + десктоп)
- 🌙 Темна / Світла тема
- 💰 Автоматичний розрахунок вартості з ПДВ (лише для адміна/бухгалтера)
- 📅 Автоматичне підтягування попередніх показань з минулого місяця

## Швидкий старт

### 1. Клонувати репозиторій

```bash
git clone https://github.com/YOUR_USERNAME/energo-oblik.git
cd energo-oblik
```

### 2. Налаштувати паролі

```bash
cp config.example.js config.js
```

Відкрийте `config.js` і замініть паролі:

```js
const APP_CONFIG = {
  users: [
    { id: 1, login: 'admin', password: 'ВАШ_ПАРОЛЬ', ... },
    ...
  ]
};
```

> ⚠️ **Важливо:** `config.js` вже додано в `.gitignore` і **ніколи не потрапить на GitHub**.

### 3. Відкрити застосунок

Просто відкрийте `index.html` у браузері — або запустіть через будь-який локальний сервер:

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

Потім відкрийте: `http://localhost:8080`

## Структура файлів

```
energo-oblik/
├── index.html          # Головний файл застосунку
├── config.js           # ❌ ЛОКАЛЬНО (паролі, не в GitHub)
├── config.example.js   # ✅ Шаблон конфігурації (без паролів)
├── .gitignore          # config.js прихований тут
└── README.md
```

## Ролі та права доступу

| Роль | Панель | Е/Е | Вода | Звіти | Фінанси | Адмін |
|------|--------|-----|------|-------|---------|-------|
| Адміністратор | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| Електрик | ✅ | ✅ (введення) | — | — | — | — |
| Директор | ✅ | ✅ (перегляд) | ✅ | ✅ | — | — |
| Бухгалтер | ✅ | ✅ (перегляд) | ✅ | ✅ | ✅ | — |

## Безпека

- Паролі зберігаються **лише локально** у файлі `config.js`
- `config.js` виключено з репозиторію через `.gitignore`
- На GitHub публікується тільки `config.example.js` із заглушками
- Застосунок повністю працює на стороні клієнта (без серверу)

## GitHub Pages

Для розгортання на GitHub Pages:

1. Зайдіть в `Settings` → `Pages`
2. Оберіть гілку `main`, папку `/ (root)`
3. Застосунок буде доступний за адресою `https://YOUR_USERNAME.github.io/energo-oblik/`

> ⚠️ На GitHub Pages `config.js` не буде (він в `.gitignore`).  
> Застосунок покаже екран налаштування — це нормально і безпечно.  
> Використовуйте застосунок **локально** з вашим `config.js`.
